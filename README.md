# Гиюр - карточки для подготовки к итоговому собеседованию

**Открыть приложение: https://dubrovai.github.io/giyur-questions/**

Веб-карточки с интервальными повторениями (по принципу Anki): 135 вопросов и ответов по итоговым темам курса гиюра - праздники, еврейский дом, цикл жизни, молитва, Танах и история, «еврейский книжный шкаф», теология. Ответы даны с позиций реформистского (прогрессивного) иудаизма, община в Израиле.

## Как пользоваться

- **Заниматься** - карточки по расписанию повторений: смотрите вопрос, вспоминаете ответ, открываете и оцениваете себя («Снова / Трудно / Хорошо / Легко»). Интервалы растут автоматически.
- Тап по разделу на главном экране - занятие только по этому разделу.
- **Все карточки и ответы** - режим просмотра с поиском.
- Прогресс хранится в браузере: у каждого, кто открыл ссылку, он свой и между устройствами не синхронизируется.
- Благословения и Шма - с ивритом, транслитерацией и переводом. Все ответы текстом: [giyur-answers.md](giyur-answers.md).

## Важно

Материал учебный и не заменяет курс: формулировки и обычаи сверяйте с материалами вашей общины и раввином. Четыре вопроса помечены «личный ответ» - в карточке только канва, ответ каждый формулирует сам.

## Как править колоду

Карточки лежат в [deck-src/](deck-src/) тремя JSON-файлами (`deck1.json` - праздники, `deck2.json` - дом и цикл жизни, `deck3.json` - молитва, Танах, теология). Поля карточки: `q` - вопрос, `a` - краткий ответ, `d` - пояснение, `he` - иврит с транслитерацией и переводом, `p` - «личный вопрос». После правки:

```
python3 deck-src/build.py
```

Скрипт пересоберёт `index.html` (приложение) и `giyur-answers.md`.
